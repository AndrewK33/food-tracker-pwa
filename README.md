# Дневник питания — PWA для iPhone

Это простое приложение без регистрации для учета калорий и БЖУ.

## Что умеет

- сканирование штрихкода камерой;
- поиск продукта через Open Food Facts;
- ручное редактирование БЖУ;
- дневная таблица питания;
- цели по калориям, белкам, жирам и углеводам;
- история по дням;
- избранные продукты;
- копирование вчерашнего дня;
- учет веса и талии;
- экспорт и импорт JSON;
- локальное хранение данных на телефоне.

## Важно

На iPhone камера в браузере работает только через HTTPS. Поэтому просто открыть `index.html` как файл обычно недостаточно.

## Как запустить без Xcode

Самый простой способ:

1. Создайте аккаунт на GitHub, если его нет.
2. Создайте новый публичный репозиторий, например `food-tracker`.
3. Загрузите туда все файлы из этой папки.
4. В настройках репозитория откройте `Pages`.
5. Source: `Deploy from a branch`.
6. Branch: `main`, folder: `/root`.
7. Откройте выданную GitHub Pages ссылку на iPhone.
8. В Safari нажмите `Поделиться` → `На экран Домой`.

После этого приложение будет запускаться как обычная иконка.

## Источник БЖУ

Приложение использует Open Food Facts. Данные открытые, но могут быть неполными или ошибочными, поэтому в приложении есть ручное редактирование.

## Конфиденциальность

Дневник, цели, избранное, вес и талия хранятся локально в `localStorage` браузера на устройстве. Регистрации и сервера приложения нет.
