Как добавить OpenAI API-ключ для увеличения точности распознавания
Самое важное

Приложение работает и без ключа. Ключ нужен только для более точного распознавания еды по фото.

  1. Открой страницу API keys.
  2. Войди в аккаунт OpenAI.
    Если аккаунта нет, зарегистрируйся на этой же странице.
  3. Нажми “Create new secret key”.
    Лучше назвать ключ понятно, например: food-tracker-phone.
  4. Скопируй ключ сразу.
    После создания его могут больше не показать полностью.
  5. Вернись сюда: ⚙️ Настройки → Распознавание по фото.
    Вставь ключ в поле “Ключ OpenAI”, включи переключатель и нажми “Сохранить настройки”.
  6. Проверь расходы.
    В OpenAI Platform стоит настроить лимит или уведомления о расходах.
Правила безопасности

Не выкладывай ключ в GitHub, Telegram, Discord, скриншоты или README. Если ключ случайно попал в интернет — удали его в OpenAI Platform и создай новый.
Как добавить продукт по фото
  1. Нажми “📸 По фото”.
    Сфотографируй продукт или выбери фото из галереи.
  2. Дождись полоски загрузки.
    При первом запуске локальная модель еды может загружаться дольше. Потом обычно быстрее.
  3. Выбери подходящий вариант.
    ИИ может ошибаться, поэтому всегда проверяй название.
  4. Введи вес.
    Приложение не угадывает вес по фото. Вес лучше взять с упаковки или кухонных весов.
  5. Проверь КБЖУ и сохрани.
    Если данных нет, введи значения вручную с этикетки.
